# S&S Ecomerce Store

React JS Mini E-Commerce Project

## Home Page
Display featured products or a simple product showcase.
## Products Page
Show all products with image, name, and price.
## Product Details Page
Show full details of a product when it is clicked.
## Cart Page
Display added products.
Show quantity and total price.
## Checkout
checkout simulation.

### Functional Requirements  
* Add products to the shopping cart.
* Remove products from the cart.
* Increase or decrease product quantity.
* Automatically calculate total cart price.
* Responsive design (works on desktop and mobile).
* Simple search bar or category filter.
* Use a JSON file or JavaScript array for product data.
* Navigation links for Home, Products, and Cart pages.
* Footer with basic information or links.
* Show simple alerts/messages (e.g., “Item added to cart”).
